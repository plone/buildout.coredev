Needed documentation changes
----------------------------

Documentation for developers should describe this feature. Something like this:

Products.ATReferenceBrowserWidget has been replaced be the API-compatible
archetypes.referencebrowserwidget. Update your imports to the new locations.

Backwards compatibility
-----------------------

The old import locations are still available and no persistent components are
involved. Backwards compatibility should not be problem.
